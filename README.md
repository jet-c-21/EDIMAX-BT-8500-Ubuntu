# EDIMAX-BT-8500-Ubuntu

Bluetooth USB setup for EDIMAX BT-8500 on Ubuntu

## All in One
```shell
cd ~ && git clone https://github.com/jet-c-21/EDIMAX-BT-8500-Ubuntu.git && \
cd EDIMAX-BT-8500-Ubuntu && \
sudo make install INTERFACE=usb && \
sudo reboot
```

## Quick Install

```shell
sudo make install INTERFACE=usb
```
